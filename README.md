# Portfolio-2

Installation et utilisation de composer
Installation de PHPMailer et DotEnv

Ajout d'un fichier .env contenant :
- SMTP_HOST => adresse du serveur
- SMTP_PORT => port utilisé pour la connexion.
- SMTP_USER => adresse email (identifiant)
- SMTP_PASSWORD => mot de passe d'application associé à l'identifiant
